# Image Denoising using Autoencoder

## Objective
To remove noise from document images using deep learning.

## Method
Used a CNN-based Autoencoder to learn mapping from noisy to clean images.

## Results
The model successfully reduces noise and improves readability of document images.

## Applications
- Document enhancement
- CCTV footage improvement
- Image restoration

## Tools Used
- Python
- TensorFlow
- OpenCV
- Jupyter Notebook
